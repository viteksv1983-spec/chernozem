Ты выступаешь как Senior Product Designer, UX/UI архитектор и Conversion Rate Optimization специалист уровня топовых digital-студий (Stripe, Linear, Apple style design).

Твоя задача — провести профессиональный аудит лендинга и предложить улучшения дизайна, не ломая текущую структуру.

Контекст проекта:

Это лендинг компании, которая продаёт экологически чистый чернозём с доставкой по Киеву и области.

Целевая аудитория:
— владельцы частных домов
— дачники
— ландшафтные дизайнеры
— строительные компании
— агробизнес

Главная цель сайта:
Максимальное количество заявок на доставку чернозема.

Ключевые действия пользователя:
1. Нажать "Замовити доставку"
2. Позвонить
3. Рассчитать стоимость доставки
4. Оставить заявку

Стиль сайта:
Premium / clean / trustworthy / agricultural / eco.

Основные цвета:
темно-зелёный
бежевый
коричневый
натуральные оттенки земли.

Шрифты:
современная антиква для заголовков
читаемый гротеск для текста.

На основе предоставленного дизайна проанализируй:

1. Hero-секцию
2. Блок преимуществ
3. Блок цен
4. Блок типов доставки
5. Блок "для кого подходит"
6. Блок "как это работает"
7. Отзывы
8. FAQ
9. Финальный CTA
10. Footer

Твоя задача:

1. Найти UX-проблемы
2. Найти слабые места конверсии
3. Проверить визуальную иерархию
4. Проверить читаемость
5. Проверить контраст
6. Проверить spacing
7. Проверить кнопку главного действия
8. Проверить мобильную адаптацию

Важно:
НЕ меняй полностью дизайн.

Работай как Senior дизайнер:
— улучши композицию
— улучши визуальную иерархию
— улучши читаемость
— усили конверсионные элементы

Сделай:

1. список конкретных UX улучшений

4. рекомендации по spacing
5. рекомендации по кнопкам CTA
6. рекомендации по typography
7. рекомендации по hero section
8. рекомендации по мобильной версии

Если возможно — предложи улучшения:

hover states

visual rhythm

НЕ усложнять дизайн.
