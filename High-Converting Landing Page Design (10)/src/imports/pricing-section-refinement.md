Your task is to refine and improve the existing pricing section in Figma for a landing page selling chernozem soil in Kyiv.

Important rules:

* Do NOT redesign the layout.
* Do NOT change the visual style of the website.
* Only improve clarity, hierarchy and conversion-focused elements.

The current section contains two product cards:

1. Bulk chernozem (main product)
2. Bagged chernozem

Goals:

1. Strengthen visual hierarchy.

Ensure the price "350 грн" is the most visually dominant element in the main card.

The structure should be:

350 грн
за 1 тонну

Price must be larger than all other text in the card.

2. Improve CTA visibility.

The button "Замовити насипом" should stand out more.

Increase contrast slightly so the button is more visible against the green background.

Do not change the color palette, only adjust brightness if necessary.

3. Improve trust signals.

Keep the line:

"Склад у Києві • 350+ доставок щомісяця"

Make it visually clear but subtle.

It should act as a trust indicator below the price.

4. Improve readability of product description.

Ensure text blocks have enough spacing and are easy to scan.

Use clear spacing between:

• title
• description
• price
• trust indicators
• bullet list
• CTA button

5. Improve the secondary card (bagged soil).

Make sure the label:

"Для клумб та городу"

is clearly visible and acts as a helpful product tag.

Do not make the secondary card stronger than the main card.

6. Maintain a premium landing page feel.

Avoid:
• unnecessary visual effects
• heavy shadows
• overly bright colors

The design should remain clean, modern and conversion-focused.

Goal:

Create a pricing section that clearly communicates the value of the bulk chernozem product while maintaining a premium and trustworthy landing page design.
