Only refine the elements described below.

• Main headline

TASK 1 — IMPROVE THE PHONE ELEMENT

Currently the phone number appears as plain text next to buttons.

This looks weak and unintentional.

Redesign it as a proper secondary CTA.

Requirements:

• Convert phone into a clickable call button
• Add phone icon
• Make it visually balanced with the other buttons
• Use outline / ghost style so it does not compete with the main CTA
• Add hover effect
• Keep it visually light but interactive

Example concept:

Primary CTA
[ Замовити доставку ]

Secondary CTA
[ Розрахувати вартість ]

Call CTA
[ 📞 Зателефонувати ]

Make sure spacing and alignment between buttons looks professional.
