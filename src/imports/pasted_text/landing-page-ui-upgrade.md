You are a **Senior Product Designer and Senior Frontend UI Architect**.

Your task is to **upgrade the visual quality of an existing landing page design to a $10k studio-level UI**, while preserving the current layout and structure.

IMPORTANT CONSTRAINTS:

• Do NOT add new sections
• Do NOT add photos or social proof blocks
• Do NOT change the structure or content
• Do NOT redesign the layout grid
• Only improve the **visual quality of existing components**

The goal is to transform the design into a **premium, modern, high-end landing page** through refined UI details.

Focus on **visual polish and design depth**.

---

FIRST — Improve visual depth

Upgrade the existing cards, blocks and surfaces to feel more premium.

Introduce subtle:

• layered shadows
• soft highlights
• subtle borders
• depth hierarchy

Example design principles:

• soft ambient shadows (large blur, low opacity)
• subtle inner highlight on surfaces
• translucent overlays

Avoid heavy shadows.

The UI should feel **modern and soft**, similar to high-end SaaS landing pages.

---

SECOND — Refine color system

Keep the existing color palette but improve the richness.

Add:

• subtle gradients
• layered background tones
• soft radial lighting effects

Hero background should include a **very subtle radial gradient lighting effect** to create visual focus around the headline.

Do NOT introduce new colors outside the current palette.

---

THIRD — Improve typography hierarchy

Refine typography without changing content.

Improve:

• heading scale
• spacing between lines
• font weight hierarchy

Guidelines:

Hero title:
large, bold, high contrast

Section titles:
clear hierarchy

Body text:
slightly reduced opacity for readability

Improve overall **typographic rhythm**.

---

FOURTH — Improve spacing and layout breathing

Increase visual elegance by adjusting spacing.

Use a consistent spacing system such as:

8 / 16 / 24 / 40 / 80 / 120

Ensure sections feel **airy and balanced**.

Avoid cramped UI.

---

FIFTH — Improve buttons and interactive elements

Upgrade the existing CTA buttons to feel premium.

Enhancements may include:

• subtle gradient fill
• soft glow shadows
• hover elevation
• smooth transitions

Buttons should feel **tactile and interactive**, not flat.

---

SIXTH — Improve card components

Existing cards should receive:

• slightly rounded corners
• soft border highlights
• layered shadow depth

Ensure cards feel **separated from the background** without looking heavy.

---

SEVENTH — Improve icon presentation

Icons should feel more polished.

Enhancements may include:

• subtle background plates
• soft translucent surfaces
• slight glow or highlight

Icons must stay simple and clean.

---

EIGHTH — Add micro-interaction hints

Design should visually imply modern UI interactions.

Add subtle hover states for:

• cards
• buttons
• icons

Use:

small lift effects
soft shadow transitions

Avoid flashy animations.

---

NINTH — Improve section separation

Enhance the perception of structure by refining background contrast between sections.

Use:

• slightly different background tones
• subtle gradients
• soft separators

Sections should feel visually distinct but cohesive.

---

TENTH — Final polish pass

Perform a final UI polish pass focusing on:

• pixel-perfect alignment
• visual balance
• consistent border radius
• consistent shadow system

The final design should feel like a **premium SaaS landing page created by a top design studio**.

Maintain the existing structure but elevate the design quality significantly.
