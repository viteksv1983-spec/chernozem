You are a senior UX designer and art director specializing in high-converting landing pages.

Your task is to refine and polish the existing hero section design in Figma for a landing page selling chernozem soil in Kyiv.

Important rules:

* Do NOT replace the background image.
* Do NOT generate a new texture.
* Only improve the visual design, readability and depth.

The background image is a realistic macro texture of fertile chernozem soil and must remain natural and detailed.

Goals:

1. Improve hero readability with a clean dark overlay gradient.

Apply the following overlay behind the text area:

linear-gradient(
90deg,
rgba(0,0,0,0.75) 0%,
rgba(0,0,0,0.55) 35%,
rgba(0,0,0,0.15) 70%
)

Remove any brown or orange tinted overlays.

2. Improve visual depth.

Add a very subtle radial highlight behind the text block to create premium visual layering:

radial-gradient(
circle at 40% 40%,
rgba(255,255,255,0.06),
rgba(0,0,0,0) 60%
)

This effect must be extremely subtle.

3. Adjust background scale.

Increase background image scale slightly (around 10–15%) so the soil texture appears stronger and more immersive.

4. Improve header design.

Replace the solid dark header with a semi-transparent header:

background: rgba(20,20,20,0.65);
backdrop-filter: blur(6px);

This should create a modern glass effect while keeping readability.

5. Maintain natural soil appearance.

Do NOT:

* blur the texture
* recolor the soil
* apply artificial filters

The soil must remain a realistic dark humus texture.

6. Typography and hierarchy.

Ensure:

* headline remains the strongest visual element
* buttons stand out clearly
* text contrast remains high on both desktop and mobile.

Design goal:

The hero should feel like a premium, high-quality landing page with a natural soil texture background, strong readability, subtle depth and modern UI polish.

Act like a senior design director polishing the final hero section before launch.
